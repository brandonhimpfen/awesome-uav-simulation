## Flight Simulators

Simulators focused on UAV flight dynamics, pilot training, and mission testing.

- [Microsoft AirSim](https://github.com/microsoft/AirSim) — Open-source simulator for drones and autonomous vehicles with realistic physics and sensor models.
- [JSBSim](https://github.com/JSBSim-Team/jsbsim) — Open-source flight dynamics model used in UAV and aircraft simulation.
- [X-Plane](https://www.x-plane.com/) — High-fidelity flight simulator with extensibility for UAV simulation.
- [RealFlight](https://www.realflight.com/) — Commercial RC and UAV flight simulator for training and testing.
- [FPV Freerider](https://fpv-freerider.itch.io/fpv-freerider) — Simulator for FPV drone flying and training.

## Physics & Robotics Simulators

General-purpose simulation environments used for UAV development and robotics.

- [Gazebo](https://gazebosim.org/) — Open-source robotics simulator with physics engines and sensor support.
- [Webots](https://cyberbotics.com/) — Robot simulation platform with support for UAVs and sensors.
- [PyBullet](https://github.com/bulletphysics/bullet3) — Physics simulation for robotics, reinforcement learning, and UAV modeling.
- [MuJoCo](https://mujoco.org/) — Physics engine for simulating complex robotic systems.
- [CoppeliaSim](https://www.coppeliarobotics.com/) — Simulation platform for robotics and UAV systems.

## AI & Reinforcement Learning Environments

Simulation environments for training UAV autonomy and intelligent systems.

- [Gym-PyBullet-Drones](https://github.com/utiasDSL/gym-pybullet-drones) — Reinforcement learning environment for quadcopters.
- [AirSim RL Environments](https://github.com/microsoft/AirSim/tree/main/PythonClient/reinforcement_learning) — RL environments built on AirSim.
- [Flightmare](https://github.com/uzh-rpg/flightmare) — High-performance simulator for drone perception and control.
- [gymfc](https://github.com/wil3/gymfc) — Reinforcement learning environment for flight control systems.
- [RotorS](https://github.com/ethz-asl/rotors_simulator) — MAV simulation framework integrated with ROS and Gazebo.

## Digital Twins & Synthetic Environments

Platforms for creating realistic environments and synthetic training data.

- [NVIDIA Isaac Sim](https://developer.nvidia.com/isaac-sim) — GPU-accelerated robotics simulator with synthetic data generation.
- [Unity Simulation](https://unity.com/products/simulation) — Platform for creating large-scale simulation environments.
- [Unreal Engine](https://www.unrealengine.com/) — Real-time 3D engine used for high-fidelity UAV simulation.
- [Omniverse](https://www.nvidia.com/en-us/omniverse/) — Platform for building digital twins and simulation pipelines.
- [Cesium](https://cesium.com/) — 3D geospatial platform for real-world environment simulation.

## Simulation Tooling & Integration

Tools and middleware for connecting simulation environments with UAV systems.

- [PX4 SITL](https://docs.px4.io/main/en/simulation/) — Software-in-the-loop simulation for PX4 autopilot.
- [ArduPilot SITL](https://ardupilot.org/dev/docs/sitl-simulator-software-in-the-loop.html) — SITL environment for ArduPilot systems.
- [MAVSDK](https://github.com/mavlink/MAVSDK) — API for controlling UAVs in simulation and real-world environments.
- [ROS](https://www.ros.org/) — Robotics middleware for integrating simulation and control systems.
- [MAVLink](https://mavlink.io/) — Communication protocol for UAV systems and simulators.

## Datasets & Synthetic Data

Datasets and tools for generating or using simulation-based UAV data.

- [AirSim Datasets](https://github.com/microsoft/AirSim) — Synthetic datasets for vision and navigation tasks.
- [VisDrone Dataset](https://github.com/VisDrone/VisDrone-Dataset) — Dataset for drone-based object detection and tracking.
- [UAVDT](https://github.com/VisDrone/UAVDT) — Dataset for UAV detection and tracking tasks.
- [Sim4CV](https://github.com/uzh-rpg/sim4cv) — Simulation environment for computer vision tasks.
- [CARLA](https://github.com/carla-simulator/carla) — Autonomous driving simulator adaptable for aerial and multi-agent environments.
